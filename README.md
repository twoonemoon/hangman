# Виселица (Hangman Game)

Учебный проект по Java, демонстрирующий принципы ООП на примере консольной
версии игры висилица.

## Как запустить
1. Открыть проект в IDE
2. Запустить класс `Main`

## Изучаемые концепции
- Классы и объекты
- Инкапсуляция
- Перечисления (enum)
- Коллекции (Set)
- Работа со строками
- GitHub
